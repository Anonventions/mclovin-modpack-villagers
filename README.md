## Mods That Add Trades to Existing Villagers

These mods add new trades to vanilla professions:

| Mod | What It Adds |
|-----|--------------|
| Farmer's Delight | Farmer/Wandering Trader: FD crops, seeds, onions |
| Oh The Biomes We've Gone | All professions: BWG items; Wandering Trader: BWG plants |
| Apotheosis | Modified villager trade system, rare trades |
| Quark | Villager trade consistency (same initial trades per profession) |

## Wandering Trader Additions

The Wandering Trader sells extra items from these mods:
- **Farmer's Delight:** Crop seeds, onions
- **Oh The Biomes We've Gone:** BWG saplings, flowers, plants
- **Alex's Caves:** Maps to Underground Cabins
- **Apotheosis:** Can spawn underground
